# Lic. Sofía Rossi · Página web

Página web de muestra para la Lic. Sofía Rossi, psicóloga clínica en Palermo, Buenos Aires.

Sitio de una sola página (landing) con atención presencial y online: ansiedad, depresión, autoestima, duelos y más.

## Cómo verla online

El sitio se publica con **GitHub Pages**. Una vez activado, queda disponible en:

`https://saltodigitalmza-maker.github.io/licrossi/`

## Archivos

- `index.html` — la página completa (HTML, CSS y JS en un solo archivo, con las imágenes incrustadas).

## Editar

Para hacer cambios, editá el archivo `index.html` y volvé a subirlo al repositorio.
